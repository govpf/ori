Ori - dev dashboard

Toutes les apps Ori, lancées en parallèle avec live-reload natif. Hard refresh inutile : chaque modif des sources rebuild la page concernée.

Légende

  • up - l'app répond sur son port
  • démarrage - l'app n'a pas encore répondu (build en cours)
  • down - l'app a planté ou a été arrêtée
