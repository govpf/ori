Mires d'authentification Keycloak

Référence visuelle des 7 écrans Keycloak rendus avec le design system Ori. Spec détaillée dans packages/docs/src/functional/Auth*.mdx.

login.ftl

Connexion

Identifiant + mot de passe + lien création de compte.

 register.ftl

Création de compte

Champs identité + email + mot de passe + confirmation.

 login-reset-password.ftl

Mot de passe oublié

Saisie d'email pour recevoir un lien de réinitialisation.

 login-update-password.ftl

Définir un mot de passe

Saisie + confirmation après réception du lien.

 login-otp.ftl

Code à usage unique

Saisie d'un code TOTP envoyé par SMS ou email.

 login-page-expired.ftl

Session expirée

Message d'expiration avec relance de la connexion.

 error.ftl

Erreur d'authentification

Page d'erreur générique avec retour vers la connexion.
